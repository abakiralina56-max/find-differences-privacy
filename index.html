Приложение Find Differences уважает конфиденциальность пользователей.

Какие данные собираются

Приложение не собирает персональные данные пользователей.

Для показа рекламы могут использоваться сервисы Google AdMob. Они могут автоматически собирать техническую информацию об устройстве, рекламный идентификатор, IP-адрес и статистику использования приложения.

Использование данных

Полученные данные используются только для показа рекламы и улучшения работы приложения.
